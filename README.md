# Machine-Learning-2-Final-Project-Kyle-Liao-100914955-
AI Resume &amp; Job Description Matching System 

## Overview
This project compares a resume and a job description using AI to determine how well they match.

## How it Works
The system uses a pretrained SentenceTransformer model to convert text into embeddings and compute similarity. It also performs keyword matching and experience-based scoring.

## Features
- Semantic similarity using neural networks
- Keyword analysis
- Resume-job alignment scoring
- Recommendations and summary output

## How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Run:
python app.py
